Transform every new tab into a personalized productivity dashboard.

Productive New Tab is a Chrome extension designed to help you stay organized and focused every time you open a new tab. It combines essential productivity tools with a clean, customizable interface.

---

## ✨ Features

- 🕒 Live clock and personalized greeting
- 🔍 Built-in Google Search
- ✅ Simple to-do list
- 📅 Google Calendar integration
- 🖼️ Dynamic Background
